# DecentralizedStorageUsingBlockchain
Uses Ganche, IPWS, AES, Advance Java, MySQL

# Before everything, replace or place the all api's and all secret keys.
# Steps to run the code
1. Compile and Deploy AccessControl.sol and FileRegistery.sol on remix IDE (Brave Browser Preferable)
2. Copy Both files Addresses and Paste it into BlockchainService.java
3. Run this command "mvn package clean"
4. Copy the Maven build package to Tomcat
5. Start the tomcat server
6. access your application
